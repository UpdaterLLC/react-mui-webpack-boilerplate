react-mui-webpack-boilerplate
=============================


## Requirements

node 5+  (recommended to use from nodebrew)


## Install && Development

Clone repository and run:

```sh
$ npm install
$ npm start
$ open http://localhost:3001
```


## Production

Build the production bundle:

```sh
$ npm run dist
```
